⚠️ Laravel 12 "Tricky" Behavior

Why prepending is critical: In Laravel 12, if a request hits a route that throws an error (401, 500, etc.) and CORS middleware isn't global, Laravel might send the error response without CORS headers. The browser sees a response without "Allow-Origin" and labels it a "CORS Error" even if the real problem was a backend crash.
